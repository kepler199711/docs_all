## 接口原型

- 请求方法：GET
- 请求地址：`https://zim-api.zego.im/?Action=MuteGroupMembers`
- 传输协议：HTTPS
- 调用频率限制：20 次/秒

## 请求示例

```json
https://zim-api.zego.im/?Action=MuteGroupMembers
&<公共请求参数>
&FromUserId=user1   
&GroupId=group
&UserIds[]=user2
&UserIds[]=user3
&Duration=100000
```
