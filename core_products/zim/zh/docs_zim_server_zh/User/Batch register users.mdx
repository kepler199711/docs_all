# 批量注册用户

- - -

## 描述

后端注册，开发者可以通过该接口批量帮用户注册 IM 账号，实现消息的接收，以便在用户登录 ZIM 后接收历史消息，常用于教育场景。

调用频率限制：20 次/秒。

## 接口原型

- 请求方法：POST
- 请求地址：`https://zim-api.zego.im/?Action=UserRegister`
- 传输协议：HTTPS

## 请求示例

- 请求地址 URL：

```json
https://zim-api.zego.im/?Action=UserRegister
&<公共请求参数>
```

- 请求消息体：

```json
{
  "UserInfo":[
    {
      "UserId": "aaa",
      "UserName": "userNamea",
      "UserAvatar": "http"
    },
    {
      "UserId": "bbb",
      "UserName": "userNameb",
      "UserAvatar": "http"
    }
  ]
}
```
