# ZIM Audio 发布日志

- - -

## 1.0.0 版本

**发布日期：2024-04-17**

ZIM Audio SDK 首次发布，支持完整的语音处理功能，包含语音采集、播放、噪声抑制（ANS）、自动增益控制（AGC）等，开发可以轻松实现高清语音消息的收发，无需关注音频处理的底层实现。若需使用，请同时搭配 ZIM SDK 接入。

| <div style={{width:"130px"}}>功能项</div>  | 功能描述 | 相关接口 |
|-------|-------|-------|
| 语言采集 | 支持录制至多 120 秒的 M4A 和 MP3 格式音频，检查录制状态。 | <ul><li>[startRecord](@startRecord)</li><li>[onRecorderStarted](@onRecorderStarted)</li><li>[onRecorderProgress](@onRecorderProgress)</li><li>[onRecorderFailed](@onRecorderFailed)</li><li>[completeRecord](@completeRecord)</li><li>[cancelRecord](@cancelRecord)</li><li>[onRecorderCompleted](@onRecorderCompleted)</li><li>[isRecording](@isRecording)</li><li>[cancelRecord](@cancelRecord)</li><li>[onRecorderCancelled](@onRecorderCancelled)</li></ul> |
| 语音播放 | 支持播放 M4A 和 MP3 格式音频，以及设置音频输出播放设备。 | <ul><li>[startPlay](@startPlay)</li><li>[onPlayerStarted](@onPlayerStarted)</li><li>[onPlayerFailed](@onPlayerFailed)</li><li>[onPlayerInterrupted](@onPlayerInterrupted)</li><li>[onPlayerEnded](@onPlayerEnded)</li><li>[setAudioRouteType](@setAudioRouteType)</li><li>[stopPlay](@stopPlay)</li><li>[onPlayerStopped](@onPlayerStopped)</li></ul> |
| 音量增益（AGC） | 支持自动调整麦克风音量，以适应远近拾音并保持音量稳定。<br />注意：如需使用本功能，请联系 ZEGO 技术支持开通，专业版和旗舰版用户可免费开通。| [enableAGC](@enableAGC) |
| 噪声控制（ANS） | 支持识别声音中的背景噪声并进行消除，开启该功能后可以使人声更加清晰。<br />注意：如需使用本功能，请联系 ZEGO 技术支持开通，专业版和旗舰版用户可免费开通。  | [enableANS](@enableANS) |
